### Usage

Download https://drive.google.com/uc?id=1rRRY-y1KdVk4UB5qhu7BjQHtfadIOmMk&export=download file and place it in models/reddit folder this will take some time 

Then 
To start server...
```sh
$ javac Server.java
$ java Server
Listening...
```
To open chat room
```sh
$ javac ChatGUI.java
$ java ChatGUI
CHATGUI OBJECT CREATED
```
